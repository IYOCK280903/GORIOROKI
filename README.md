# GORIOROKI
Aplikasi mobile GoRioRokiEvents adalah sebuah Event Management System yang dirancang untuk memberikan kemudahan dalam melihat, memantau, dan mengelola berbagai acara (event) secara efisien melalui perangkat smartphone.  Proyek ini dikembangkan oleh tim Rio dan Roki sebagai bagian dari pemenuhan tugas akhir Pemrograman Mobile.
